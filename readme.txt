nik.im for wordpress allows comments on your blog to be modified so all links are shortened to nik.im urls.

== Description ==
This plugin automatically changes all urls in comments to nik.im short urls. This helps in your efforts to avoid link spam and also helps in SEO.


== Installation ==

Activating the nik4wp plugin is simple! Follow these 2 steps!

1. Upload `nik4wp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
